# calibrate-longevity

Turn your bloodwork, DNA, and health data into a ranked longevity plan.

## Install

Unzip (or copy) this folder into your AI agent's skills directory:

| Agent | Put it here (create if missing) |
|---|---|
| **Claude Code** | `.claude/skills/calibrate-longevity/` |
| **Cline / Roo Code** | `.claude/skills/calibrate-longevity/` |
| **MiMoCode / OpenCode** | `.agents/skills/calibrate-longevity/` |
| **Cody** | `.cody/skills/calibrate-longevity/` |
| **Cursor** | `.cursor/rules/calibrate-longevity/` |
| **Codex CLI** | `.codex/skills/calibrate-longevity/` |

Or use the one-liner (requires [npx skills](https://skills.sh)):

```
npx skills add your-org/your-repo@calibrate-longevity -g
```

## Use it

Start a session with your agent and say:

> `/calibrate-longevity`

Then paste in your bloodwork, DNA reports, and health goals. The agent will walk you through the rest.

## What you'll need

- Recent bloodwork (any format)
- DNA report (23andMe, Ancestry, etc. — optional)
- Health tracker data (optional)
- What you want to optimize for

## What you'll get

A ranked plan with specific dosages, protocols, and re-check cadences — calibrated to your data and risk tolerance.
