# Cognition compound reference

All entries below are T2–T3. General caveats apply.

---

## Semax

| Property | Detail |
|---|---|
| Mechanism | Synthetic ACTH(4–10) analogue. Increases BDNF, NGF, and TrkB receptor expression. Modulates dopamine and serotonin systems. |
| Typical dose | 400–800 µg/day intranasal (standard), 800–1600 µg/day (high dose) |
| Half-life | ~15 minutes (metabolites active for hours) |
| Onset | 15–30 minutes, builds over 3–7 days |
| Tolerance | Low; moderate tolerance over 4–6 weeks, resolve with 1–2 week break |
| Reported effects | Focus, verbal fluency, mild anxiolysis, reduced brain fog |
| Side effects | Rare: mild headache, nasal irritation |
| Stack notes | Synergistic with noopept (different mechanisms). Add PQQ for mitochondrial support. |
| Tier | T3 |

## Noopept

| Property | Detail |
|---|---|
| Mechanism | Prolyl endopeptidase inhibitor. Modulates AMPA and NMDA receptors, increases NGF and BDNF. |
| Typical dose | 10–30 mg 2–3× daily oral, or 10–20 mg intranasal |
| Half-life | ~20 minutes (active metabolite cycloprolylglycine: ~3–5 hours) |
| Onset | 15–30 minutes oral, 5–10 minutes intranasal |
| Tolerance | Rapid tolerance reported (3–7 days). Cycle 2 days on / 1 day off or 4 weeks on / 1 week off. |
| Reported effects | Working memory, pattern recognition, sensory clarity |
| Side effects | Headache (especially at >30 mg), irritability, short-term memory blunting at high doses |
| Stack notes | Often stacked with semax. Pair with choline source (alpha-GPC, citicoline) to prevent headache. |
| Tier | T3 |

## Dihexa

| Property | Detail |
|---|---|
| Mechanism | Angiotensin IV analogue, hepatocyte growth factor (HGF) activator. Promotes synaptogenesis via c-Met. Crosses BBB. |
| Typical dose | 10–35 mg/day oral (sublingual or transdermal may be more bioavailable) |
| Half-life | ~15 hours (long-acting) |
| Onset | 2–4 weeks for noticeable effect |
| Tolerance | Not well documented |
| Reported effects | Cognitive recovery after brain injury or sleep deprivation, memory consolidation |
| Side effects | Very limited human data. Potential: hypotension, dizziness. |
| Stack notes | Long cycle (8–12 weeks). Monitor BP. |
| Tier | T3 |

## Phenylpiracetam

| Property | Detail |
|---|---|
| Mechanism | Racetam family. Modulates AMPA, nicotinic acetylcholine, and dopamine D1/D2 receptors. Mild NDRI activity. |
| Typical dose | 100–200 mg 1–2× daily (AM only — can disrupt sleep) |
| Half-life | ~3–5 hours |
| Onset | 30–60 minutes |
| Tolerance | Rapid, pronounced tolerance (significant on day 3–4 if used daily). Limit to 1–2× per week. |
| Reported effects | Physical energy, mental stamina, cold resistance, mild euphoria at first use |
| Side effects | Insomnia, overstimulation, headache, elevated BP |
| Stack notes | Pair with choline source (CDP-choline, alpha-GPC). Avoid afternoon dosing. |
| Tier | T3 |

## Modafinil / Armodafinil

| Property | Detail |
|---|---|
| Mechanism | Dopamine reuptake inhibitor (weak). Also affects orexin/hypocretin, histamine, and norepinephrine systems. |
| Typical dose | 50–200 mg modafinil / 75–150 mg armodafinil (AM only) |
| Half-life | Modafinil: ~12–15 hours. Armodafinil: ~15 hours. |
| Onset | 30–60 minutes |
| Tolerance | Minimal with 1–2×/week use. Daily use leads to tolerance within 2–4 weeks. |
| Reported effects | Wakefulness, sustained attention, executive function, reduced impulsivity |
| Side effects | Insomnia, headache, dry mouth, appetite suppression, anxiety. Rare: Stevens-Johnson syndrome. |
| Stack notes | Avoid afternoon dosing. L-theanine (200 mg) can blunt anxiety. Hydrate well. |
| Tier | T2 (prescription in many countries) |
